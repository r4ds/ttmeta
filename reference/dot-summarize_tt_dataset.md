# Summarize a TidyTuesday dataset

Summarize a TidyTuesday dataset

## Usage

``` r
.summarize_tt_dataset(tt_dataset, dictionary, year, week)
```

## Arguments

- tt_dataset:

  The dataset to summarize.

- dictionary:

  The extracted data dictionary for this dataset.

- year:

  The year of this dataset (to include in the summary info).

- week:

  The week of this dataset (to include in the summary info).

## Value

A `tbl_df` summarizing the data.
